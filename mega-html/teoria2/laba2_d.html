Теория к лабораторной работе:
Регистрация входящего документа


Вниз

Прежде чем приступать к работе с документом, его необходимо ввести в систему, т.е. зарегистрировать. При регистрации формируется регистрационная карточка (РК) документа, в которую заносятся сведения о документе и присваивается регистрационный номер, формируемый по единым для данной подгруппы документов правилам.

Система «ДЕЛО» позволяет осуществлять регистрацию входящих, исходящих и внутренних документов: письма, приказы, договоры, обращения граждан и прочее.

При регистрации формируется регистрационная карточка документа, в которую заносятся сведения о документе и присваивается регистрационный номер, формируемый по единым для данной подгруппы документов правилам. Каждый тип РК имеет свой реквизитный состав, основанный на требованиях нормативных документов и делопроизводственной практике (у исходящих и внутренних документов он одинаков). К карточке могут быть прикреплены файлы, содержащие тексты или графические образы самих документов.

Электронная регистрационная карточка состоит из двух экранов. Первый экран содержит основные реквизиты РК, второй - дополнительные. Переход от раздела к разделу осуществляется с помощью закладок, расположенных в правом верхнем углу. В нижней части основного экрана отражаются относящиеся к этому документу поручения.

Система позволяет в автоматизированном режиме регистрировать переданные по электронной почте документы, в том числе снабженные электронной цифровой подписью. Для документов, которые готовятся с помощью MS Word, также реализована упрощенная процедура регистрации прямо из привычной для пользователя среды редактора.

Для облегчения и ускорения процедуры регистрации используются следующие инструменты:

  • задание автоматически заполняемых значений реквизитов (по умолчанию);
  • заполнение реквизитов путем выбора нужного значения из заранее настроенного справочника;
  • перечень наиболее часто используемых стандартных текстов;
  • установление ссылок на другие документы;
  • отнесение документа к тематическим рубрикам и др.

1) Войдите в функцию «Регистрация документов»

Рис. 1 - Функция «Регистрация документов»

2) Выберите группу документов, соответствующую регистрируемому документу

Рис. 2 - Окно групп документов

3) Установите (если необходимо) Дату РК
Дату можно ввести с помощью календаря (календарь вызывается двойным щелчком левой клавишей мыши или одинарным щелчком правой) или вручную.

4) Нажмите кнопку «Выбрать»

Откроется окно основного раздела РК, в которой по умолчанию заполнены поля:
- Рег № - регистрационный номер документа, заполненный автоматически по шаблону, заданному для выбранной группы документов
- От – дата регистрируемого документа
- Экз.№ - номер экземпляра оригинала документа
- Доступ - гриф доступа пользователей к содержимому карточки; по умолчанию установлено значение «общий»

Рис. 3 - Регистрационная карточка входящего документа

Заполните поля, содержащие информацию об авторах документа и регистрационных реквизитах документа
 5.1) Поле «Корреспондент»
Введите название корреспондента с клавиатуры (значение имеет синий цвет), нажмите кнопку «Enter».
Если такая организация уже зарегистрирована, то ее название приобретет черный цвет и проверка справочника на этом закончится.
В противном случае откроется окно «Выбор из справочника организации». Добавьте организацию в справочник:
Добавление новой организации в справочник в процессе регистрации
Находясь в окне «Выбор из справочника организации»
- Нажмите кнопку «Добавить организацию» справа в углу на панели инструментов. Откроется окно для ввода реквизитов организации.
- В верхнем поле выберите папку, в которую будет записана новая организация.
- Внесите необходимые сведения об организации.
- Нажмите кнопку «Записать». Название организации отобразится в справочнике.

Рис. 4 - Окно с информацией об организации-корреспонденте

Нажмите кнопку «Выбрать». При этом наименование организации в поле «Корреспондент» РК примет черный цвет.

5.2) Заполните открытые поля:
«Исх. №» - исходящий номер поступившего документа
«Дата» - дата отправки документа, заполняется вводом с клавиатуры или двойным щелчком
«Подписал» (при необходимости) – ФИО должностного лица, подписавшего документ

6. Проверьте РК на повторность
После заполнения полей «Корреспондент», «Исх. №», «Дата» и «Подписал» нажмите кнопку или сочетание клавиш Ctrl+F10.
Если передвижение по данным полям осуществляется с помощью табулятора [Tab], проверка РК на повторность осуществляется автоматически.

7.Заполните поле «Состав» (количественные характеристики состава документа: количество листов, наличие и количество приложений в формате «количество листов основного документа »,«количество листов приложения»)

8.Заполните поле «Доставка» (наименование способа получения документа).

9.Заполните поле «Кому» (наименование отдела или ФИО и должность работника, который будет рассматривать документ) выбором из справочника или вводом с клавиатуры.

Справочник открывается с помощью стрелки справа от строки «Кому». Для заполнения поля с клавиатуры необходимо ввести ФИО и нажать клавишу «Enter».

10. Заполните поле «Содержание» (краткое описание содержания документа)
Для использования текстовых заготовок войдите в перечень стандартных текстов с помощью клавиши Insert, выделите нужную строку, нажмите кнопку «Выбрать». Вставленный текст при необходимости можно отредактировать.

Порядок добавления стандартных текстов:
Находясь в окне «Выбор текста из списка»:
Для добавления нового списка:
- Нажмите кнопку «Редактировать перечень списков» справа от наименования списка;
- Нажмите кнопку «Добавить список» ,
- В открывшееся окно внесите название списка,
- Нажмите кнопку «Записать»,
- Нажмите кнопку «Закрыть»,
Название списка можно отредактировать, войдя в режим редактирования - кнопка
Для добавления нового стандартного текста в выбранный список:
- Нажмите кнопку «Редактировать перечень текстов» справа от поля с перечнем текстов (или F7 на клавиатуре);
- Нажмите кнопку «Добавить текст» для внесения стандартного текста,
- В открывшемся окне внесите текст в текстовое поле,
- Нажмите кнопку «Записать»

11. В поле «Рубрикатор» внесите вид регистрируемого документа (если регистрируемая группа подразделяется на виды)
Нажмите на кнопку . В предлагаемом списке выделите необходимое значение, нажмите кнопку «Выбрать». Нажмите кнопку «ОК».

12. При необходимости добавьте комментарии в поле «Примечания» (поле для ввода произвольного текста комментария с характеристиками документа или, возможно, способа его обработки)

13. Сохраните изменения с помощью кнопки и закройте РК или продолжите регистрацию с помощью кнопок

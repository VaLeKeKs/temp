Вниз

Прежде чем приступать к работе с документом, его необходимо ввести в систему, т.е. зарегистрировать. При регистрации формируется регистрационная карточка (РК) документа, в которую заносятся сведения о документе и присваивается регистрационный номер, формируемый по единым для данной подгруппы документов правилам.

Система «ДЕЛО» позволяет осуществлять регистрацию входящих, исходящих и внутренних документов: письма, приказы, договоры, обращения граждан и прочее.

При регистрации формируется регистрационная карточка документа, в которую заносятся сведения о документе и присваивается регистрационный номер, формируемый по единым для данной подгруппы документов правилам. Каждый тип РК имеет свой реквизитный состав, основанный на требованиях нормативных документов и делопроизводственной практике (у исходящих и внутренних документов он одинаков). К карточке могут быть прикреплены файлы, содержащие тексты или графические образы самих документов.

Электронная регистрационная карточка состоит из двух экранов. Первый экран содержит основные реквизиты РК, второй - дополнительные. Переход от раздела к разделу осуществляется с помощью закладок, расположенных в правом верхнем углу. В нижней части основного экрана отражаются относящиеся к этому документу поручения.

Система позволяет в автоматизированном режиме регистрировать переданные по электронной почте документы, в том числе снабженные электронной цифровой подписью. Для документов, которые готовятся с помощью MS Word, также реализована упрощенная процедура регистрации прямо из привычной для пользователя среды редактора.

Для облегчения и ускорения процедуры регистрации используются следующие инструменты:

  • задание автоматически заполняемых значений реквизитов (по умолчанию);
  • заполнение реквизитов путем выбора нужного значения из заранее настроенного справочника;
  • перечень наиболее часто используемых стандартных текстов;
  • установление ссылок на другие документы;
  • отнесение документа к тематическим рубрикам и др.

Поиск документов
Система "ДЕЛО" предоставляет широкие возможности поиска зарегистрированных документов, проектов документов, а также поручений.

Поиск может осуществляться как по значению отдельных реквизитов, так и по произвольному их сочетанию. Помимо этого в системе реализован поиск по текстам документов (прикрепленных к РК файлов). Сложность поисковых запросов не ограничена.

При необходимости сложный поисковый запрос может быть сохранен и использован неоднократно. Отобранные в результате поиска документы можно хранить в персональных папках пользователей или распечатать в виде перечня.

В зависимости от прав пользователь может осуществлять поиск документов в пределах одной из доступных ему картотек или во всем множестве зарегистрированных в системе документов.

Можно также задать поиск для целых групп документов за определенный период времени. Механизмы рубрикации документов (отнесения к определенной теме) и их связывания между собой позволяют быстро, точно и без дополнительных усилий находить все документы по определенной тематике или вопросу и при необходимости легко разбираться в истории проблемы.

7.1 Поиск документов
Откройте основное окно системы «Дело». Выберите функцию «Поиск».

Рис. 1 – Функция «Поиск»

Откроется окно, в левой части которого перечислены стандартные варианты поиска по реквизитам РК (по умолчанию открыт общий поисковый запрос, первый по списку).

Рис. 2 - Окно функции «Поиск»

Каждый из представленных запросов соответствует определенному виду РК и содержит свои собственные поля, по которым можно искать РК.
- Если необходимо, то смените вид запроса. Для этого выделите строку с нужным наименованием поискового запроса.
- Заполните поля, по которым будет производиться поиск.
- Нажмите кнопку «Начать поиск» .В результате будет выдан в виде таблицы список искомых РК, удовлетворяющих введенному запросу. Отобранные РК можно просматривать и редактировать.

Рис. 3 - Окно с результатами поиска

- Чтобы просмотреть и отредактировать отобранные РК, выделите в таблице интересующие РК и нажмите кнопку «Просмотреть РК» (или дважды щелкните левой кнопкой мыши по нужной РК). Перемещаясь с помощью указателей со стрелками «Предыдущая РК» и «Следующая РК» по выбранным РК, можно их просмотреть и, если необходимо, внести изменения, используя кнопку «Изменить РК - F7».
- Для перехода на поиск по резолюции (поручению) и поиск проектов документов, нажмите соответствующую кнопку .Дальнейшие действия аналогичны поиску по РК.

В системе реализован также механизм быстрого поиска и открытия регистрационной карточки документа. Вызов окна для быстрого поиска осуществляется с помощью горячих кнопок Shift+F. Окно быстрого поиска используется для задания требуемого варианта поиска - по номеру, заданному вручную, или с использованием ручного сканера.

7.2. Личные папки
Бывает, что отобранные РК удобно объединить в одну папку, где их всегда можно было бы найти, не составляя больше сложных запросов.
- Находясь в окне «Отобранные документы», нажмите кнопку «Папки» . Откроется окно «Папки пользователя».
- Чтобы завести новую папку, нажмите кнопку «Добавить».
- Заполните поля «Наименование» и, если требуется, снабдите Вашу папку комментариями.
- Нажмите кнопку «Записать». Созданная папка появится в списке папок пользователя.
- Выделите РК (одну или несколько), которые Вы хотели бы поместить в созданную папку.
- Выберите пункт меню «Управление» => «Добавить в». Откроется окно «Папки пользователя».
- Поставьте указатель на нужную папку и нажмите кнопку «Выбрать». Система скопирует отобранные РК в папку и вернется в окно «Отобранные документы».
- Если надо добавить еще РК в папку, выберите РК, нажмите кнопку меню «Управление» => «Добавить в», из предложенных папок выберите нужную и нажмите кнопку «Выбрать».
Добавлять РК документов в личные папки можно из функций «Поиск», «Кабинеты», «Регистрация», используя команду «Добавить в личную папку».

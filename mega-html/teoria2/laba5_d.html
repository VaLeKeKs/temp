Теория к лабораторной работе:
Регистрация обращений граждан


Вниз

Прежде чем приступать к работе с документом, его необходимо ввести в систему, т.е. зарегистрировать. При регистрации формируется регистрационная карточка (РК) документа, в которую заносятся сведения о документе и присваивается регистрационный номер, формируемый по единым для данной подгруппы документов правилам.

Система «ДЕЛО» позволяет осуществлять регистрацию входящих, исходящих и внутренних документов: письма, приказы, договоры, обращения граждан и прочее.

При регистрации формируется регистрационная карточка документа, в которую заносятся сведения о документе и присваивается регистрационный номер, формируемый по единым для данной подгруппы документов правилам. Каждый тип РК имеет свой реквизитный состав, основанный на требованиях нормативных документов и делопроизводственной практике (у исходящих и внутренних документов он одинаков). К карточке могут быть прикреплены файлы, содержащие тексты или графические образы самих документов.

Электронная регистрационная карточка состоит из двух экранов. Первый экран содержит основные реквизиты РК, второй - дополнительные. Переход от раздела к разделу осуществляется с помощью закладок, расположенных в правом верхнем углу. В нижней части основного экрана отражаются относящиеся к этому документу поручения.

Система позволяет в автоматизированном режиме регистрировать переданные по электронной почте документы, в том числе снабженные электронной цифровой подписью. Для документов, которые готовятся с помощью MS Word, также реализована упрощенная процедура регистрации прямо из привычной для пользователя среды редактора.

Для облегчения и ускорения процедуры регистрации используются следующие инструменты:

  • задание автоматически заполняемых значений реквизитов (по умолчанию);
  • заполнение реквизитов путем выбора нужного значения из заранее настроенного справочника;
  • перечень наиболее часто используемых стандартных текстов;
  • установление ссылок на другие документы;
  • отнесение документа к тематическим рубрикам и др.

1) Войдите в функцию «Регистрация документов»

Рис. 1 - Функция «Регистрация документов»

2) Выберите группу документов «Обращения граждан»

3) Установите (если необходимо) Дату РК
Дату можно ввести с помощью календаря (календарь вызывается двойным щелчком левой клавишей мыши или одинарным щелчком правой) или вручную.

4) Нажмите кнопку «Выбрать»

Рис. 2 – РК «Обращения граждан»

5) Заполните поле «Гражд.»
Введите ФИО с клавиатуры (значение имеет синий цвет), нажмите кнопку Enter. Если такой гражданин уже обращался, то его ФИО приобретет черный цвет, и проверка справочника на этом закончится. При этом система выдаст список документов, полученных от данного гражданина ранее. Просмотрите предыдущие документы.
а) Если обращение повторное:
- из предложенного списка откройте РК первичного обращения, после чего закройте текущую РК, не сохраняя.
- зарегистрируйте повторное обращение, как связанное, в группе «Повторные обращения», тип связки «Первичный – Повторный»

Регистрация документа, как связанного, с копированием реквизитов из первичной РК:
- Найдите документ, с которым надо связать новый регистрируемый документ, в папке кабинета или с помощью функции «Поиск». - Двойным щелчком откройте РК найденного документа. - Нажмите на кнопку «Создать связанную РК» . В открывшемся окне: - Установите указатель напротив нужной группы документов.
- Установите нужный вид связки между связываемыми документами.
- В нижней части окна отметьте галочками реквизиты, которые могут быть скопированы в регистрируемый документ.
- Нажмите кнопку «Выбрать».
- В результате откроется РК нового документа с частично заполненными реквизитами.
- Если необходимо, отредактируйте реквизиты РК.
- Введите информацию в незаполненные поля РК согласно правилам регистрации данной группы документов.
- Перенесите предложенный системой номер на бумажный документ.
- Сохраните РК или продолжите регистрацию

б) Если обращение является обращением по тому же вопросу:
- из предложенного списка откройте РК первичного обращения, после чего закройте текущую РК, не сохраняя.
- нажмите на кнопку .
- нажмите кнопку «Добавить сопроводительный документ» .
- выберите из справочника организацию, из которой поступило обращение.
- внесите сведения о сопроводительном документе (содержание, исходящий номер, дата).

в) Если обращение не повторное, вернитесь в созданную РК и продолжите регистрацию:
Если данный гражданин ранее не обращался, откроется окно «Выбор из справочника Граждане». Для добавления нового гражданина в справочник
- Нажмите кнопку для создания новой записи. Откроется окно для ввода реквизитов гражданина.
- Внесите необходимые сведения о гражданине.
- Нажмите кнопку «Записать». Фамилия гражданина отобразится в справочнике.
- Нажмите кнопку «Выбрать».

6) Заполните поле «Кому» (кто из руководства будет рассматривать данное обращение).

7) Заполните поле «Состав» (количественные характеристики состава документа: количество листов, наличие и количество приложений в формате «количество листов основного документа », «количество листов приложения»).

8) Заполните поля «Содержание» (краткое описание содержания документа)
Для использования текстовых заготовок войдите в перечень стандартных текстов с помощью клавиши Insert, выделите нужную строку, нажмите кнопку «Выбрать». Вставленный текст при необходимости можно отредактировать.

9) В поле «Доставка» укажите способ доставки обращения

10) Заполните поле «Адресаты»
- Нажмите кнопку
- Выберите кнопку «Добавить организацию» или «Добавить гражданина» или «Добавить внутреннего адресата» .
- Выделите адресата в справочнике (используйте окно «Найти» для быстрого поиска нужного наименования).
- Нажмите кнопку «Выбрать».
- В окне «Адресаты:….» нажмите кнопку для отметки отправки. Выберите дату и способ отправки.
- Нажмите кнопку «ОК».

11) При необходимости добавьте комментарии в поле «Примечания»

12) Прикрепите к РК файл с текстом документа в поле «Файлы».
Нажмите крайнюю левую кнопку окна «Файлы» . Откроется окно «Выберите файл для привязки».Найдите требуемый файл в Вашей файловой системе, выделите его и нажмите кнопку «Открыть». Произойдет возврат в РК, в окне «Файлы» появится ссылка на документ. Содержание прикрепленного файла можно просмотреть, дважды кликнув по его иконке в окне «Файлы».

13) Сохраните изменения с помощью кнопки и закройте РК или продолжите регистрацию с помощью кнопок

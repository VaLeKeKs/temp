Теория к лабораторной работе:
Работа в кабинете


Вниз

Прежде чем приступать к работе с документом, его необходимо ввести в систему, т.е. зарегистрировать. При регистрации формируется регистрационная карточка (РК) документа, в которую заносятся сведения о документе и присваивается регистрационный номер, формируемый по единым для данной подгруппы документов правилам.

Система «ДЕЛО» позволяет осуществлять регистрацию входящих, исходящих и внутренних документов: письма, приказы, договоры, обращения граждан и прочее.

При регистрации формируется регистрационная карточка документа, в которую заносятся сведения о документе и присваивается регистрационный номер, формируемый по единым для данной подгруппы документов правилам. Каждый тип РК имеет свой реквизитный состав, основанный на требованиях нормативных документов и делопроизводственной практике (у исходящих и внутренних документов он одинаков). К карточке могут быть прикреплены файлы, содержащие тексты или графические образы самих документов.

Электронная регистрационная карточка состоит из двух экранов. Первый экран содержит основные реквизиты РК, второй - дополнительные. Переход от раздела к разделу осуществляется с помощью закладок, расположенных в правом верхнем углу. В нижней части основного экрана отражаются относящиеся к этому документу поручения.

Система позволяет в автоматизированном режиме регистрировать переданные по электронной почте документы, в том числе снабженные электронной цифровой подписью. Для документов, которые готовятся с помощью MS Word, также реализована упрощенная процедура регистрации прямо из привычной для пользователя среды редактора.

Для облегчения и ускорения процедуры регистрации используются следующие инструменты:

  • задание автоматически заполняемых значений реквизитов (по умолчанию);
  • заполнение реквизитов путем выбора нужного значения из заранее настроенного справочника;
  • перечень наиболее часто используемых стандартных текстов;
  • установление ссылок на другие документы;
  • отнесение документа к тематическим рубрикам и др.

1) Войдите в функцию «Кабинеты»

Рис. 1 – Функция «Кабинеты»

2) Откройте папку «Поступившие», проверьте поступление новых документов (выделены зеленым цветом).

РК можно сортировать. Для сортировки элементов перечня РК (РКПД) используются заголовки столбцов, которые одновременно являются кнопками сортировки. Щелчок на заголовке-кнопке приводит к сортировке записей перечня в соответствии со значениями данной графы. При этом сортировка производится по алфавиту (от А до Я) для буквенных значений и в порядке возрастания для числовых.

Рис. 2 – Папка «Поступившие» Кабинета

а) Попробуйте сортировать РК по разным полям.
б) Определите, в каком порядке располагаются РК при щелчке на заголовке графы, содержащей идентификаторы групп документов (сортировка документов по типу РК для папок 1-5).

Если не требуется выводить на экран полный перечень документов, поступивших в папку, можно с помощью фильтра отобрать для показа в рабочей области папки только те записи, которые удовлетворяют определенным условиям. Настройка фильтра:
- выберите меню Управление - Фильтр, Настройка фильтра или щелкните на кнопке, расположенной на панели инструментов
- выберите из окна Список фильтров (окно откроется, если уже существуют фильтры) подходящий фильтр или создайте новый фильтр, заполнив поля с реквизитами РК в окне Настройка фильтра в соответствии с критериями отбора
- сохраните фильтр
- для применения щелкните по кнопке Применить

в) Определите с помощью фильтра, есть ли в папке документы, плановая дата исполнения которых уже наступила

3. Зарегистрируйте в папке новый документ.
Чтобы зарегистрировать новый документ в папке кабинета нужно выделить в папке кабинета любую строку и выбрать команды Управление - Просмотреть РК. Откроется окно выбранной РК. Выберите команды Файл - Создать РК или щелкните на соответствующей кнопке панели инструментов. Откроется окно выбора группы документов. Создайте и зарегистрируйте РК (без поручений).

4. Отправьте созданный документ в папку «Принять к исполнению»
Для этого выделите РК, кликните по выделенному правой кнопкой мыши и в появившемся меню выберите «Принять к исполнению».

5. Внесите в РК контрольную резолюцию (поставленную а контроль), исполнителем которой будет сотрудник вашего отдела, а также сотрудник отдела, который установлен на соседнем компьютере. Плановая дата исполнения – день, следующий за днем выполнения лабораторной работы.
Для этого:
- Найдите РК в папке Кабинета (1 или 2). Откройте РК, дважды кликнув по строке.
- Нажмите кнопку «Ввести резолюцию» на панели инструментов раздела «Резолюция». Если необходимо, чтобы вводимая резолюция автоматически снималась с контроля ответственным контролером нажмите стрелку возле кнопки на панели управления резолюциями, выберите пункт «Ввести подчиненную резолюцию», в окне ввода резолюции, установите флаг в окне «Снимать с контроля каскадно».
- Заполните поле «Содержание резолюции» путем введения текста с клавиатуры или путем выбора стандартного текста при нажатии клавиши «Insert». Вставленный текст при необходимости можно отредактировать.
- Если необходимо поставить резолюцию на контроль, введите плановую дату исполнения. Когда заполнено поле «Плановая дата исполнения», запись об РК документа будет автоматически помещена в папку «На контроле» Кабинета автора (контролера) резолюции.
- Назначьте исполнителей резолюции: введите фамилию исполнителя в поле «Добавить» и нажмите «Ввод» (для добавления нескольких исполнителей повторите действие).
- Для контрольной резолюции выберите контролера путем ввода фрагмента фамилии в поле «Контролер» или путем выбора из справочника (контролером является автор резолюции).
- Нажмите «Записать» в окне ввода резолюции. После этого РК будет автоматически разослана в Кабинеты исполнителям в папку «Поступившие», если резолюция контрольная - в Кабинет автора резолюции в папку № 3 «На контроле».

6. Сформируйте и отправьте напоминание.
Для этого:
- Найдите в папке «На контроле» документ, к которому в предыдущем пункте была создана резолюция
Если напоминание должно быть отправлено нескольким исполнителям одного поручения, выведите на экран полный перечень записей, поступивших в папку (параметр Исполнитель в фильтре должен иметь значение «все исполнители»).
- Выберите из меню Управление -Отправить напоминание. Откроется окно: в поле Текст введите текст напоминания.
- При необходимости, того или иного исполнителя поручения можно удалить из списка адресатов. Для этого выделите его фамилию в нижней части окна и щелкните на кнопке Удалить.
- Чтобы отправить напоминание, щелкните на кнопке Отправить. Окно закроется, произойдет возврат к окну текущей папки.

7. Просмотрите напоминания и отметьте факт ознакомления с напоминанием (в папке «На исполнении» и в папке «На контроле»; обратите внимание на отличия).
Для этого:
- Выделите РК, помеченное флагом напоминания
- Правой клавишей мыши вызовите меню с командами управления
- Выберите команду «Прочитать напоминания» (в папках «Поступившие» и «На исполнении») или «Просмотреть напоминание» (в папке «На контроле»). Откроется окно со списком напоминаний
- Выделите напоминания, с которыми ознакомились
- Нажмите на кнопку «Отметить прочтение», после чего отметка в графе «Флаг напоминания» исчезнет

8.После исполнения документа, введите сведения об исполнении в РК документа в зависимости от варианта закрытия
а)Если исходящим документом закрывается только одна из контрольных резолюций, а входящий документ находится дальше на исполнении, выполните п.I,II,III,IV
б)Выполните п. I, III- IV, если документ не требует подготовки исходящего документа, а ответ передается по телефону или устно; входящий документ направлен для сведения; исходящий документ регистрируется в ОДО.

I. Найдите и откройте РК исполненного документа (неисполненные документы находятся в папке «На контроле»)
II. Спишите входящий документ в дело путем заполнения Журнала передачи в РК.
- Находясь в окне РК, щелкните на кнопке «Журнал передачи» или выберите из меню Реквизиты - Журнал передачи документа (или нажмите сочетание клавиш Ctrl + J). Если кнопки управления вверху окна не активны, нажмите кнопку «Редактировать» в правой нижней части окна.
- Нажмите кнопку «Добавить запись о списании документа в дело».
- В открывшемся справочнике номенклатуры дел выделите номер дела, в которое подшивается документ.
- Нажмите кнопку «Выбрать»
- Укажите, оригинал или копия документа подшивается в дело с помощью кнопок (копия) или (оригинал).
III.Укажите дату исполнения резолюции в управлении. (неисполненные документы находятся в папке «На контроле»)
- Перейдите на резолюцию руководителя управления стрелкой в поле «Резолюция».
- Нажмите кнопку «Редактировать резолюцию».
- В окне редактирования резолюции введите фактическую дату исполнения документа
- Нажмите «Записать»
IV. Исполните документ путем введения отчета исполнителя.
- В открытой РК перейдите на резолюцию руководителя стрелкой в поле «Резолюция»
- Нажмите кнопку «Ввести отчет» на панели управления окна «Резолюция». (Если РК закрыта, в списке РК выделите нужную. Выберите «Управление» => «Исполнить»).
- Введите дату исполнения.
- При необходимости введите отчет исполнителя (можно использовать стандартные тексты с помощью клавиши «Insert»)
- Нажмите «ОК».
Исполненные контрольные документы остаются в папке «На исполнении», отмеченные датой исполнения, до момента централизованного снятия с контроля.

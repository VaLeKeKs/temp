Зачем таможеннику знания
в IT сфере?

Информационные технологии уверенно вошли во все сферы жизни человека.
Одна из основных целей, при которой используются IT в таможенном деле - ускорение и повышение качества взаимных действий таможни и участников ВЭД.

После обучения вы приобретете:

– знание базовых языков программирования: HTML, XML/XSL, Java Script;
– знание основ web-разработки
